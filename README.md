# Sistema Workflow

Sistema moderno de gestión de procesos, automatización operativa y control de flujos de trabajo desarrollado para optimizar la administración de tareas, seguimiento de procesos y coordinación organizacional.

---

## ⚙️ Instalación Local

### 1. Clonar el repositorio

```bash
git clone https://github.com/rdmezalazo/sistema-workflow.git
```

### 2. Ingresar al proyecto

```bash
cd sistema-workflow
```

### 3. Instalar dependencias

```bash
npm install
```

### 4. Configurar variables de entorno

Crear archivo `.env`:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 5. Ejecutar en desarrollo

```bash
npm run dev
```
